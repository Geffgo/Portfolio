Fiche Interactive Donjons & Dragons (JDR)

Application Web Front-End - 17 avril 2026

Le Projet

Développement d'une interface web interactive permettant de gérer dynamiquement les fiches de personnages D&D. L'application centralise les compétences (Barbare, Paladin), les sorts magiques et les emplacements de sorts pour améliorer l'organisation du jeu.

Architecture

Application bâtie sur un triptyque HTML/CSS/JavaScript pur, avec une logique encapsulée dans une fonction auto-exécutée pour garantir la maintenabilité du code.

Objectifs Techniques

  • Concevoir une interface SPA sans rechargement de page.
  • Structurer les données sous forme d'objets JavaScript (nom, résumé, description).
  • Filtrage dynamique des sorts par niveau.

Compétences acquises

Manipulation du DOM (createElement, innerHTML) JavaScript ES6+ (Tableaux d'objets, filtres) Gestion des événements (addEventListener) Conception UI/UX (Système d'accordéon) Interactivité et Filtrage dynamique
