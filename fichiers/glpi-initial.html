Mission 1 : Déploiement initial de GLPI v10

Documentation de Mission - 4 février 2026

Présentation

L'objectif est d'installer GLPI dans un environnement de test fonctionnel avec une sécurité de base. Cette étape fondamentale permet d'anticiper les futurs besoins de haute disponibilité sous forme de cluster.

Objectifs

  • Déployer une infrastructure LAMP sur Debian 12.
  • Configurer une base MariaDB sécurisée en utf8mb4.
  • Mettre en place des sauvegardes automatisées via Cron.

Compétences acquises

Analyse & Choix architectural Administration Système Linux Gestion de bases de données SQL Sécurisation applicative Automatisation de maintenance
